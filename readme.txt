Hello this application pulls data based on the value searched in the python app named sql.py

Steps to configure
1. Set a search text inside of sql.py
2. run scripts on MySQL server create a database nbc
3. create table nbcdemo using script given. It will create a simple table with two column values. First coulumn is key(url) assiciated and second column is description where the searched data is saved related to this page.
4. click on sql.py file and it will execute and give result for NBC as I have used "NBC universal" as keyword for search.
5. click on 20_urls_results.py and it will fetch first 20 results from the keyword search and fetch you the data from each of those pages.
